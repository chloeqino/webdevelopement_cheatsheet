Responsive Design

To create a responsive page, add the line of code inside the head tag:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

when the view point is #px or smaller:
@media only screen and (max-width: #px)
{
/* your code here*/
}

When width < height:
@media only screen and (orientation: portrait)
{
/* your code here*/
}
